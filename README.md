# l-host

A local-first web-based file manager and media server.  
Run it on any machine and access your files from a browser.

## Features

- Browse, upload and manage files & folders
- Stream audio / video with a full-featured player
- Image viewer with HEIC support
- PDF viewer
- Archive (.zip) preview
- Category views: Audio, Videos, Images, Files, Archives, APKs

## Requirements

- Node.js 18+

## Setup

```bash
npm install
npm start
```

Then open `http://localhost:5000` in your browser.
